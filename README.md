# :construction_worker: Como rodar
```bash
# Clone o Repositoria
$ git clone https://github.com/joaofnetojfn/Proffy_030820_WEb.git

### 💻 Rode o Project Web

```bash
# Vá para a pasta web
$ cd Proffy/web

# Instale as depedencias
$ yarn install

# Rode a aplicação
$ yarn start
```
Acesse: http://localhost:3000/ para ver o resultado.
